# Notes

- Simple no-fuss Markdown support.
- Local storage of notes.
- Minimal UI.
- HTML preview.
- Zen mode.


## Supported markdown formattings

- **Italic** with `*Italic*`.
- **Bold** with `**Bold**`.
- **Italic bold** with `***Italic bold***`.
- **Bullet list** with `- Bullet list`.
- **Headings** with `# Heading`, `## Heading 2`, `### Heading 3`, `#### Heading 4`.
